# ⚽ Fut Training App

## 📌 Descripción
Aplicación desarrollada en Java con interfaz gráfica (Swing) para la gestión de entrenamientos deportivos.

Permite registrar, editar y consultar entrenamientos y deportistas, utilizando persistencia de datos en archivos locales.

---

## 🚀 Funcionalidades
- Registro de entrenamientos
- Gestión de deportistas
- Clasificación por categorías
- Edición de información
- Persistencia automática en archivos `.txt`

---

## 🛠️ Tecnologías utilizadas
- Java
- Swing (Interfaz gráfica)
- Manejo de archivos (File I/O)

---

## 📂 Estructura del proyecto
- `/src/Fut_Training_app` → lógica de negocio
- `/src/Graficas` → interfaces gráficas
- Recursos gráficos incluidos

---

## 💾 Persistencia de datos
El sistema genera automáticamente los archivos `.txt` necesarios al ejecutar la aplicación, por lo que no es necesario incluirlos manualmente.

---

## ▶️ Cómo ejecutar
1. Abrir el proyecto en NetBeans o cualquier IDE compatible con Java
2. Ejecutar la clase `Principal.java`
3. La aplicación iniciará y creará los archivos necesarios automáticamente
